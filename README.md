# Notes App - React Version (with Cardano transaction support)

A beautiful notes application with a yellow pad paper design, built with React.  
Includes notes app functionality and Cardano transaction integration.

## Features

- Create, view, edit, and delete notes
- Tag notes with: To-Do, Reminder, Work, or School
- Filter notes by tag
- Notes are saved to localStorage
- Beautiful handwritten-style design with yellow pad paper aesthetic
- Cardano blockchain transaction support

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
    ```
    npm install
    ```
2. Start the development server:
    ```
    npm start
    ```
    The app will open at `http://localhost:3000`
